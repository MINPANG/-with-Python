1. 명령어 프롬프트 열기!

2. 처음 깃허브 설정!
git --version // 깃허브 잘 깔렸나 확인 ! 

3. 깃허브 설정하고 싶은 working directory로 이동 
cd 경로 입력

4. git config --global user.name "MINPANG" // 내 깃허브 아이디 입력

5. git config --global user.email "https://github.com/MINPANG/-with-Python.git"  // 연결하고자하는 깃허브 URL주소

// 처음 한번만
6. git init // 연결 후 폴더 생성(초기화) 설정! 

// 처음 한번만 
7. git remote add origin https://github.com/MINPANG/-with-Python.git // 원격저장소를 git에 등록

// 처음 한번만
8. git push -u origin master //github 아이디와 패스워드 입력하면 파일 정상적으로 업로드!
   (컴퓨터 포맷하면 해야겠지?)
   git pull origin master // 이건 git에 있는거 가져오는거!

9. 커밋 순서!!
git add "파일명"
git commit -m "뭘 변경햇는지 작성"
git push



git restore "파일명"  //깃허브에 올라가 있는 파일로 되돌리기 
git push // 파일들 깃허브로 밀어넣기
